# AI-PathFinder - STEMConnect

Sistema de orientación vocacional con IA que conecta estudiantes latinoamericanos con científicos STEM.

## Características

- **Inicio**: Presentación del sistema
- **Intereses**: Exploración de áreas STEM
- **Orientación**: Chatbot vocacional con IA
- **Visualización**: Dashboard con datos reales
- **Inspiración**: Científicos latinoamericanos
- **Recursos**: Contenido educativo generado con IA

## Tecnologías

### Frontend
- React 18 + Vite
- Tailwind CSS
- Lucide React (iconos)

### APIs y Datos
- OpenAlex API (datos académicos)
- Wikipedia API
- Wikidata SPARQL
- Semantic Scholar
- Hugging Face API (chatbot IA)

### Scraping y Extracción
- Python + Requests
- BeautifulSoup
- Scrapy
- Selenium/Playwright

### Procesamiento NLP
- spaCy
- Transformers
- Sentence-BERT
- Hugging Face Transformers

### Análisis de Datos
- Pandas + NumPy
- Matplotlib + Seaborn
- gender-guesser + Genderize.io

### Generación de Contenido IA
- Llama 3 + Mistral
- Stable Diffusion XL + Flux
- Diffusers

### Base de Datos Vectorial
- ChromaDB / Qdrant

### Servicios AWS
- AWS Amplify (hosting y deploy)
- AWS Bedrock (modelos de IA)
- AWS Lambda (funciones serverless)
- AWS API Gateway (APIs REST)
- AWS S3 (almacenamiento)

### Infraestructura
- Google Colab
- HuggingFace Spaces
- Git + GitHub
